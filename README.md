# 科杰科技营销作战地图

面向销售、售前与管理层的区县级营销作战平台。系统以中国华东和西南区域为重点，统一管理政府、产业、平台、单体项目与交付合作伙伴五类资源。

## 已实现能力

- 作战地图：全国全览、省市区县下钻、五类资源图层、项目检索、阶段与风险筛选、项目详情。
- 项目工作台：表格、看板、日历三种视图，保存视图、批量操作、CSV 导出、新增与编辑项目。
- 经营分析：管道预测、资源结构、区域金额、销售漏斗、负责人排名。
- 数据管理：模板下载、CSV 导入预检、质量检查、数据源与导入记录。
- 提醒中心：红黄风险分级、逾期和停滞规则、批量处理。
- 权限管理：管理员、售前、销售三类角色；销售仅能查看本人数据。
- 系统配置：用户、角色权限、销售阶段与提醒规则管理。
- 三级地图：华东、西南支持省、市、区县逐级下钻和面包屑返回。
- 省级聚合与点位避让：全国/大区按省汇总项目，圆圈大小表示预计成交金额；下钻后自动展开同区县项目点。
- 政府资源牵线单位：政府项目录入、导入、台账、地图筛选与颜色图例保持一致。
- 生产后端：Supabase Auth、PostgreSQL、RLS、审计日志和管理员邀请用户。

## 本地运行

```bash
npm ci
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

配置真实后端后的生产构建：

```bash
cp .env.example .env.production
npm run build:production
```
